University Management Portal - Role Based Frontend

This project is made only with:
- HTML
- Basic CSS

Main idea:
- Student panel and instructor panel are separated.
- Student cannot see instructor options.
- Instructor cannot see student panel options.
- Assignment create page and assignment submission page are different.

Pages:
- index.html
- register.html

Student Pages:
- student-dashboard.html
- student-courses.html
- student-assignments.html
- student-attendance.html
- student-announcements.html
- student-forum.html

Instructor Pages:
- instructor-dashboard.html
- instructor-students.html
- instructor-courses.html
- instructor-create-assignment.html
- instructor-submissions.html
- instructor-attendance.html
- instructor-announcements.html
- instructor-forum.html
- instructor-reports.html

How to run:
1. Extract the zip file.
2. Open index.html in browser.
3. Click login as student or login as instructor.

This is a beginner friendly frontend demo for university portal project 11.
